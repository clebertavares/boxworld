BOX WORLD v1.0.0
* Introdução
	Remake do jogo clássico box world desenvolvido com o objetivo de mostrar
	o uso da pyGame para desenvolvimento de jogos simples.

	Necessário que a pyGame esteja instalada para que possa executar. Visite: http://www.pygame.org/

* O jogo
	Execute o interpretador python no arquivo BoxWorld.pyw
	- Comandos
		Direcionais - movem o personagem
		R - Reinicia o mapa
		W - Passa do mapa
		Ctrl + Z - Desfaz

* Gerando executável 
	Necessário que py2exe esteja instalada. Visite: http://www.py2exe.org/
	Execute no interpretador python o arquivo setup.py
	Ao fim do processo o executável se encontrará na pasta dist
	Copie a pasta file para pasta dist
